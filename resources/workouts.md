# Running Workout Library

Reference these session types when designing training. Apply judgement —
this library captures established patterns, not every possible session.
Cross-reference zones.md for pace/HR targets.

---

## Session Types

| Type | Structure | Pace/Zone | When to use |
|------|-----------|-----------|-------------|
| Recovery | 30–45min very easy | Z1, E+30s/km | After hard days, illness recovery |
| Aerobic | 45–75min steady | Z2, E pace | 60–70% of weekly volume |
| Long Run | 90min–3hr | Z2, start easy | Weekly — primary endurance session |
| Medium-Long | 60–90min | Z2, moderate finish | Mid-week Pfitzinger staple |
| Tempo | 20–40min continuous | Z4, T pace | Build phase, 1x/week |
| Cruise Intervals | 4–6 x 1600m, 60–90s jog | T pace | Threshold development |
| VO2max Intervals | 5–6 x 1000m, 400m jog | I pace | Peak phase, 1x/week max |
| Repetitions | 8–12 x 400m, 400m jog | R pace | Speed, neuromuscular |
| Progression Run | Final 25–33% at faster pace | E → M → T | Race simulation, discipline |
| Strides | 6–8 x 20–30sec, full recovery | Fast but controlled | 2x/week, after easy runs |
| Fartlek | Unstructured speed play | Varies | Mental break, squad sessions |

## Sample Sessions

**Threshold Cruise Intervals**
```
Warm-up: 15min easy + 4x100m strides
Main: 5 x 1600m @ T pace, 90s jog recovery
Cool-down: 10min easy
Total: ~75min
```

**VO2max Development**
```
Warm-up: 15min easy + 6x100m strides
Main: 5 x 1000m @ I pace, 400m jog recovery
Cool-down: 15min easy
Total: ~60min, high stress — 48–72hr recovery needed
```

**Progression Long Run (Marathon-Specific)**
```
Total: 24–28km
- First 60%: E pace (Z2)
- Middle 25%: M pace (Z3)
- Final 15%: T pace (Z4)
Practice race-day fuelling every 30–45min
```

**Standard Progression Run**
```
16km total:
- 0–6km: Easy (Z2)
- 6–10km: Steady (Z2–3)
- 10–14km: Tempo (Z4)
- 14–16km: Threshold (Z5a)
Teaches pace discipline and finishing strong
```

## Strides — Evidence Base

Strides are 20–30 second bursts at 85–90% effort added after easy runs.
Research shows:
- ~2% running economy improvement after just 40 days
- ~3.2% performance gain
- Minimal injury risk — short enough not to add meaningful load
- Neuromuscular benefit without fatigue cost

How to do them: Gradual acceleration for 20–30 seconds to 85–90% effort
(not a sprint), walk back for 60–90 seconds full recovery, repeat 4–8 times.
Add to easy runs 2–3x/week. Not after hard sessions.

## Individual Response Variation

Not all athletes respond the same way to training stimuli. Research suggests:
- ~31.5% respond best to polarised training (very easy + very hard, minimal moderate)
- ~31.9% respond best to pyramidal (more threshold work, less extremes)
- ~18.7% respond well to both
- Remainder respond better to other distributions

This justifies agreeing an approach per block rather than applying a fixed
methodology. If a block isn't producing expected adaptation, changing the
distribution is a legitimate lever — not a failure.

## Intensity Distribution (80/20 Rule)

| Zone | % of Weekly Volume | Purpose |
|------|--------------------|---------|
| Z1–2 (Easy) | 75–80% | Aerobic development, recovery |
| Z3 (Moderate) | 5–10% | Race-specific only, use sparingly |
| Z4–5 (Hard) | 15–20% | Adaptation stimulus |

Avoid excessive Z3 "grey zone" training. Easy should be easy, hard should
be hard. Most athletes naturally gravitate to Z3 — actively resist this.

## Strength & Injury Prevention

1–2 sessions/week.

**Core (every session):**
- Plank 3x45–60s
- Side plank 2x30s each
- Dead bug 2x12 each side
- Bird dog 2x10 each side

**Running-specific:**
- Calf raises 
- Single-leg Romanian deadlifts
- Lunges
- Hip circles / glute activation

**Timing:**
- Strength after easy sessions, not before key sessions
- 6+ hours from quality run sessions if doubling
- Taper to maintenance (2 sets, light) in race week
