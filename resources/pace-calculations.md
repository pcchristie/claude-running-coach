# Pace Calculations & Training Zones

## Determining Training Paces

### From Race Times (Preferred)
Use Daniels VDOT method. Current paces in the athlete's ATHLETE.md — recalculate after any key race. Quick estimation from threshold (T) pace:

| Zone | Offset from T pace |
|------|-------------------|
| Easy (E) | T + 60–70 sec/km |
| Marathon (M) | T + 15–25 sec/km |
| Threshold (T) | — baseline |
| Interval (I) | T − 15–20 sec/km |
| Repetition (R) | T − 25–35 sec/km |

Race time equivalents (same fitness level):
- 5K × 2.1 ≈ 10K
- 10K × 2.17 ≈ Half marathon
- Half × 2.11 ≈ Marathon

### From Conversational Pace (No Race Data)
Easy pace = conversational pace. From there, estimate:
- Tempo: Easy − 45–60 sec/km
- Interval: Easy − 2:30–3:30/km

### Talk Test (Simplest)
| Effort | Can you talk? | Zone |
|--------|--------------|------|
| Very easy | Full sentences, comfortable | Z1 Recovery |
| Easy | Full sentences | Z2 Aerobic |
| Moderate | Short phrases, some effort | Z3 Tempo |
| Hard | Few words only | Z4 Threshold |
| Very hard | 1–2 words max | Z5 Interval |
| Max | Cannot speak | Z6 Repetition |

## Pace Adjustments

### Heat (Most Relevant — Perth Climate)

| Temperature | Adjustment |
|-------------|-----------|
| <18°C | No adjustment |
| 18–24°C | Slow by 5–15 sec/km |
| 24–30°C | Slow by 15–30 sec/km |
| 30–35°C | Slow by 30–45 sec/km |
| >35°C | Slow significantly or reschedule |

Add extra adjustment for high humidity (>70%). In extreme heat, run by HR
and RPE — abandon pace targets entirely. Hydration status amplifies heat effect.

### Terrain

| Surface | Adjustment |
|---------|-----------|
| Flat road | Baseline |
| Rolling hills | +10–20 sec/km average |
| Trail (easy) | +10–30 sec/km |
| Trail (technical) | +30–90 sec/km |
| Significant hills | Run by effort, ignore pace |

On hilly or technical terrain, use time and effort as the primary metric.

### Altitude
- Every 1,000m elevation: ~3–5% slower pace
- Allow 2–3 weeks for partial adaptation

### Fatigue State
| State | Adjustment |
|-------|-----------|
| Well-rested | Hit prescribed paces |
| Moderate fatigue | Easy pace +10–20 sec/km |
| High fatigue | Recovery run or rest |
| 24–48hr post hard workout | Easy pace +20–30 sec/km |

## Common Pacing Mistakes

**Running easy days too hard** — the most common error. Most runners drift to
Z3 when they should be Z2. The fix: slow down until it feels almost
embarrassingly easy. If in doubt, go slower.

 **Running Hard Days Too Easy** - Not hitting appropriate intensity on workout days. **Fix**: If prescribed tempo pace, run actual tempo pace (not slower). **Why it matters**: Stimulus insufficient for adaptation.

**Racing every workout** — chasing a good pace on tempo days. Training is not
racing. Save effort for actual races.

**Ignoring conditions** — forcing prescribed pace in 30°C heat. Run by effort
when conditions warrant.

## Communicating Paces in Plans

Always include effort description alongside the number. Better to say:

> "5km easy — conversational pace, full sentences comfortable. Based on your
> current fitness, roughly 5:10–5:35/km, but go slower if the effort feels
> harder than that. Heat adds 15–30 sec/km."

Than:

> "5km at 5:20/km"

The effort description is what the athlete should actually target. The pace
is a guide, not a constraint.
