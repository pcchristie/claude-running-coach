# Periodization & Progressive Overload

## Macrocycle Structure

Divide total training time into phases. Percentages are guidelines —
adjust based on current fitness, injury history, and time available.

| Phase | % of Time | Focus |
|-------|-----------|-------|
| Base | 40–50% | Aerobic capacity, volume, technique |
| Build | 30–40% | Intensity, race-specific work, peak volume |
| Peak/Specific | 10–15% | Race-pace work, course-specific prep |
| Taper | 1–3 weeks | Volume down, some intensity maintained |

---

## Phase Detail

### Base Phase
- Mostly Z1–Z2 (80/20 distribution, Lydiard principles)
- Volume increases gradually — see progressive overload below
- Strides 2x/week for neuromuscular stimulus without real load
- No threshold or VO2max work until aerobic base is solid
- Strength training 2x/week

### Build Phase
- Introduce tempo and threshold work (1x/week each)
- Long runs may include M-pace segments
- Peak volume weeks occur here
- 3:1 or 4:1 loading cycles (3 weeks build, 1 week recovery)
- Strength reduces to 1x/week

### Peak/Race-Specific Phase
- Volume reduces slightly, quality maintained
- Race-pace work becomes primary quality session
- Long runs shorter but more specific (progression runs)
- Practice race nutrition in long sessions

### Taper
- Volume reduction: 40–60% below peak
- Maintain some intensity — don't go completely easy
- Short strides to keep legs sharp
- Focus on sleep, nutrition, mental prep
- Duration: 2–3 weeks for marathon; 1 week for shorter races

---

## Microcycle (Weekly) Principles

More important than which specific day — these principles must hold:

1. **48hr between quality sessions** in the same sport
2. **Hard days hard, easy days easy** — no grey zone drift
3. **Easy day after long runs** — long run recovery is non-negotiable
4. **Consistent structure** — same general pattern each week
5. **Fixed commitments** — plan the week around the athlete's recurring obligations


### Microcycle Design
Build each week from first principles using the constraints above — fixed
commitments, recovery requirements, and session spacing. Don't start from
a template; start from what the athlete needs this specific week.

## Progressive Overload

### Foundation vs. Form

See `resources/assessment.md` for the full foundation vs current form framework
and validation protocol. Always run that assessment before setting progression
rates. The table below assumes the assessment has been done.

| Athlete status | Progression rate | Rationale |
|----------------|-----------------|-----------|
| True beginner | 5–10%/week | Everything is new; high injury risk |
| 1–2 years experience | 10%/week | Standard |
| Strong foundation, rebuilding | 10–15%/week | Body remembers faster |
| Recent high fitness | Maintain or small build | Focus on specificity |

Injury-related time off requires more caution than time off due to life
circumstances.

### Loading Cycles
- Standard: 3 weeks build, 1 week recovery (volume −30–40%)
- Injury-prone or older athletes: 2 weeks build, 1 week recovery
- Recovery weeks are not optional — fitness is built during rest

### Long Run Progression (Marathon Example)
```
Wk 1: 16km → Wk 2: 18km → Wk 3: 20km → Wk 4: 14km (recovery)
Wk 5: 20km → Wk 6: 22km → Wk 7: 24km → Wk 8: 16km (recovery)
...continue to 32–35km peak
```
Never increase long run by more than 2–3km per week.

## Marathon-Specific Build (16–20 Weeks from Base)

**Weekly structure at peak:**
- 5–6 running days
- 70–90km/week for sub-3 target
- One long run (28–35km)
- One quality session (tempo or intervals)
- One medium-long run (18–22km)
- Remaining: easy

**CTL target for sub-3 marathon: 55–65 by race week.**
Reference athlete's current CTL from ATHLETE.md — significant building required if starting from a low base.

## Physiological Adaptation Timelines

| System | Visible changes | Optimised |
|--------|----------------|-----------|
| Neuromuscular coordination | 1–2 weeks | 4–6 weeks |
| Lactate clearance | 2–4 weeks | 6–8 weeks |
| Aerobic enzymes | 3–4 weeks | 8–12 weeks |
| Mitochondrial density | 4–6 weeks | 12–16 weeks |
| Capillarisation | 6–8 weeks | 16–24 weeks |
| Cardiac adaptations | 8–12 weeks | 24+ weeks |
| Bone/tendon strength | 12–24 weeks | 1–2 years |

**Implication:** Base phases should be longer than athletes want. Intensity
phases shorter. There are no shortcuts for the deepest adaptations — and
tendon/bone adapt slowest, which is why injury risk spikes when CV fitness
outpaces structural readiness.
